# startpage

A minimal browser startpage/homepage with quick-access bookmarks.

## Features

- Lightweight HTML/CSS with no JavaScript
- Dark theme with smooth hover transitions
- Responsive flexbox layout
- Keyboard accessible with focus indicators

## Usage

1. Clone or download this repository
2. Edit `index.html` to customize your bookmarks
3. Set `index.html` as your browser's homepage

## Customization

Edit the `<li>` items in `index.html` to add, remove, or modify bookmarks:

```html
<li>
    <a
        href="https://example.com"
        target="_blank"
        rel="noopener noreferrer"
        >Example</a
    >
</li>
```

Modify `style.css` to change colors, fonts, or layout.
